# Hotel Staff Web Dashboard

A Next.js web application for hotel staff operations management.

## Features

- **Dashboard**: Overview of daily stats (check-ins, check-outs, pending requests)
- **Service Requests**: Manage and track guest requests
- **Room Management**: Visual grid of room statuses
- **Housekeeping**: Track cleaning tasks and priorities

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
