# Karma by ReZ - Standalone Expo App

Standalone mobile app for the Karma gamification feature.

## Setup

```bash
npm install
npx expo prebuild
npx expo run:android
npx expo run:ios
```

## Screens

### Consumer
- `/login` - Authentication
- `/karma/home` - Main hub
- `/karma/my-karma` - Passport & history
- `/karma/explore` - Event listing
- `/karma/event/[id]` - Event detail
- `/karma/missions` - Missions
- `/karma/micro-actions` - Micro actions
- `/karma/leaderboard` - Rankings
- `/karma/wallet` - Wallet
- `/karma/scan` - QR scanner
- `/karma/communities` - Communities
- `/karma/communities/[slug]` - Community detail

### Admin
- `/admin` - Dashboard
- `/admin/karma-score` - KarmaScore admin
- `/admin/perks` - Perk management
