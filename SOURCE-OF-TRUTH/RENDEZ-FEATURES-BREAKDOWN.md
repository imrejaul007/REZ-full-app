# WHAT IS RENDEZ?

**Rendez** is an **Intent-Based Social Matching Platform** where people find others to do activities with - not just date.

> **"Whatever you want to do — find the right person to do it with."**

# TWO USER MODES

## Mode 1: 🔴 BOOK FIRST (High Intent)
> "I already booked. Now I need a partner."

**Flow:**
```
1. User books activity (dinner, sports, etc.)
2. Creates plan: "Booked dinner @ 8PM - need 1 person"
3. Others apply to join
4. User selects approved applicants
5. Meetup happens
6. QR check-in validates
7. Karma updated
```

**Why Powerful:**
- Zero timepass users
- High seriousness
- Direct merchant revenue

---

## Mode 2: 🟢 MATCH FIRST (Scale)
> "I want to do something. Let's find someone first."

**Flow:**
```
1. User creates: "Looking for dinner partner tonight"
2. Others see and apply
3. Match happens
4. THEN booking via REZ
5. QR check-in validates
6. Karma updated
```

**Why Needed:**
- Low friction entry
- High volume growth
- Solves cold start problem

# KARMA SYSTEM (TRUST INFRASTRUCTURE)

## How Karma Works

| Signal | Karma | Reliability Impact |
|--------|-------|-------------------|
| Shows up (QR verified) | +5 | +0.01 |
| Good feedback | +3 | +0.005 |
| No cancellation | +1 | - |
| No-show | -15 | -0.05 |
| Last-minute cancel | -10 | -0.03 |
