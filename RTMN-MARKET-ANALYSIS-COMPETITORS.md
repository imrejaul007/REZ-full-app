# PART 4: KEY DIFFERENTIATORS

---

## Why ReZ Wins

### 1. Merchant-First Model
```
Zomato/Swiggy: Merchant pays 20-30% → HIGH COST
ReZ: Merchant pays 0% platform fee → LOW COST + AI tools free
```

### 2. Unified Loyalty (Karma)
```
Other platforms: Fragmented rewards
ReZ: One karma score, one coin, ALL merchants
```

### 3. AI Moat (REZ Intelligence)
```
No competitor has:
- Local commerce intent graph
- Multi-vertical AI (restaurant + hotel + retail)
- Offline-to-online attribution
- Real-time personalization
```

### 4. Ecosystem Flywheel
```
Employee downloads ReZ app
  ↓
Uses Consumer App, Rendez, Do App, Habixo
  ↓
Generates data for REZ Intelligence
  ↓
Better recommendations → More engagement
  ↓
Merchants get more customers
  ↓
More merchants join
  ↓
More employees onboard via CorpPerks
```

### 5. Corporate Lock-in
```
One CorpPerks client = 1,000 employees
Each employee uses multiple ReZ products
Hard to switch ecosystem
High LTV per corporate client
```
