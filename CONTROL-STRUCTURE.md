# CONTROL STRUCTURE

## Core Principle

**RTMN Digital (Holding)** owns everything strategic.
Each **Operating Company** controls their domain.
**REE** controls all money/data flows.

| **Investments | RTMN Digital | M&A, funding |
| **Billing/Invoicing | Rabtul | GST, invoices |
| **P&L per company | Each company | Own P&L |
| **Coin Economics | REE | Minting, burning |

### Financial Flows

```
RTMN Digital (Treasury)
 │
 ├── Monitors all company P&L
 ├── Allocates budget
 ├── Raises funding
 └── Owns equity

Each Company (P&L)
 ├── Revenue - Costs = Profit
 ├── Pays RTMN (IP fee)
 └── Pays Rabtul (infra fee)
 ├── Pays REZ Intelligence (AI fee)
 └── Pays RTMN Finance (payment fee)
```
