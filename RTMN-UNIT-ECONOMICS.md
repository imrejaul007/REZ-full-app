# MERCHANT UNIT ECONOMICS

## REZ Commerce (Restaurant Merchant)

### Acquisition Cost (CAC)

| Channel | Cost | Conversion |
|---------|------|------------|
| Outbound sales | ₹1,500/merchant | 15% |
| Inbound (website) | ₹500/merchant | 25% |
| Referral (existing) | ₹200/merchant | 40% |
| Marketing (ads) | ₹800/merchant | 10% |
| **Weighted Average** | **₹800/merchant** | **20%** |

### Lifetime Value (LTV)

| Metric | Value | Notes |
|--------|-------|-------|
| Avg monthly GMV | ₹50,000 | Per restaurant |
| Platform fee | 1% | ₹500/month |
| Subscription | ₹999/month | SaaS revenue |
| Avg tenure | 24 months | Industry avg |
| **LTV** | **₹35,976** | (₹500 + ₹999) × 24 |

### LTV Calculation

```
LTV = (Monthly Revenue) × (Average Tenure)
    = (₹500 transaction + ₹999 subscription) × 24 months
    = ₹1,499 × 24
    = ₹35,976
```

### Payback Period

| Metric | Value |
|--------|-------|
| CAC | ₹800 |
| Monthly revenue | ₹1,499 |
| Payback | 0.5 months |

### LTV:CAC Ratio

```
LTV:CAC = ₹35,976 : ₹800
        = 45:1
```

**Excellent ratio!** But need to account for:
- Support costs
- Payment processing
- Infrastructure
- Discounts

### Adjusted LTV (After Costs)

| Item | Monthly | Annual |
|------|---------|--------|
| Revenue | ₹1,499 | ₹17,988 |
| (-) Support cost | ₹50 | ₹600 |
| (-) Infra cost | ₹20 | ₹240 |
| (-) Payment cost | ₹150 | ₹1,800 |
| Net Contribution | ₹1,279 | ₹15,348 |

```
Adjusted LTV = ₹15,348
Payback = 0.6 months
LTV:CAC = 19:1
```

# PAYMENT TRANSACTION ECONOMICS

### Per Transaction

| Item | ₹1,000 transaction |
|------|---------------------|
| Gross revenue | ₹3 | (0.3%) |
| (-) Payment gateway | ₹2 | (0.2%) |
| (-) Settlement cost | ₹0.20 | |
| Net revenue | ₹0.80 | |

### Volume Economics

| Transactions/month | Gross | Net |
|-------------------|-------|-----|
| 1,000 | ₹3,000 | ₹800 |
| 10,000 | ₹30,000 | ₹8,000 |
| 1,00,000 | ₹3,00,000 | ₹80,000 |
| 10,00,000 | ₹30,00,000 | ₹8,00,000 |

### Break-even

```
Fixed costs (per month):
- Team: ₹5,00,000
- Infrastructure: ₹2,00,000
- Compliance: ₹3,00,000
Total: ₹10,00,000

Break-even: ₹10,00,000 / ₹0.80 = 12.5 lakh transactions/month
```
