## 3-LAYER STRUCTURE

```
LAYER 1: RTMN DIGITAL (HOLDING)
 │
 ├── Owns: Core IP, Trademarks, Governance
 │
 ▼
LAYER 2: RABTUL TECHNOLOGIES (SHARED INFRA)
 │
 ├── Owns: AI/ML, Auth, Payments, APIs, Cloud, DevOps
 ├── Powers: ALL operating companies
 │
 ▼
LAYER 3: OPERATING COMPANIES (6)
 │
 ├── ReZ Commerce (Consumer + Merchant)
 ├── ReZ Intelligence Labs (AI - THE MOAT)
 ├── ReZ Media Network (Ads + Loyalty)
 ├── StayOwn Hospitality (Hotels + Smart Living)
 ├── CorpPerks (Enterprise + Restaurant SaaS)
 └── RTMN Finance (Payments - Separate due to RBI)
```

## COMPANY 1: REZ COMMERCE TECHNOLOGIES
### (FLAGSHIP - Main Growth Company)

Consumer and merchant commerce platform with QR codes.

| Product | Description |
|---------|-------------|
| Consumer App | Mobile app for end-users (ordering, payments, coins) |
| Merchant App | Mobile app for merchants (POS, CRM, analytics) |
| Do App | Discovery & engagement app |

| Web Menu | Menu browsing |

- SaaS subscriptions
- Ad revenue share

**Competitors:** Zomato, Swiggy, Shopify

## COMPANY 2: REZ INTELLIGENCE LABS
### (THE MOAT - AI Company)

**What it does:**
AI and data infrastructure that powers ALL other companies.

| Product | Description |
|---------|-------------|
| REZ Mind | Unified AI brain |
| Intent Graph | Track user behavior (search, view, cart, checkout) |
| Targeting Engine | Ad targeting based on intent |
| Personalization | Recommendations for each user |
| Attribution | Multi-touch tracking (who bought because of which ad) |
| Copilot | AI assistant (Consumer, Merchant, Support) |

**Revenue Model:**
- AI SaaS licensing to other companies
- External API fees
- Attribution fees

**Why it's the moat:**
- Owns ALL user data
- Owns ALL AI models
- Cannot be replicated by competitors

**Competitors:** Google AI, Meta AI, IBM Watson

## COMPANY 3: RABTUL TECHNOLOGIES
### (SHARED INFRASTRUCTURE)

**What it does:**
"Internal AWS + Stripe" that powers all companies.

| System | Description |
|--------|-------------|
| API Gateway | Routing, rate limiting |
| Auth Service | JWT, OTP, sessions |
| Search Service | Full-text search |
| Notifications | Push, email, SMS |
| Wallet Service | Coins, loyalty |
| Payment Service | UPI, Razorpay |
| Database | MongoDB, Redis |
| DevOps | CI/CD, monitoring |

**Revenue Model:**
- Per-API call fees (₹0.001-0.01/call)
- Platform licensing (₹10K-1L/month)
- Enterprise contracts

**All other companies PAY Rabtul to use this infrastructure.**

## Why Split Like Tata?

| Investor Concern | Solution |
|----------------|----------|
| Who owns the AI? | ReZ Intelligence Labs |
| Who owns the data? | RTMN Digital |
| What if one fails? | Isolated risk |
| Can IP leak? | Centralized IP |
| Clean cap table? | Separate entities |
