# CONFIDENCE SCORING

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                       CONFIDENCE SCORING                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Score = (Intent × 0.35) + (Context × 0.30) + (History × 0.25) + (Load × 0.10)   │
│                                                                          │
│   ┌─────────────────────────────────────────────────────────────────┐    │
│   │                                                                 │    │
│   │   Intent Match (35%)                                          │    │
│   │   ├─────────────────────────────────────────────────────────┤  │    │
│   │   │ How well does the intent match agent capabilities?       │  │    │
│   │   │ "Order food" → Culinary Expert: 95%                     │  │    │
│   │   │ "Order food" → Fitness Expert: 20%                       │  │    │
│   │   └─────────────────────────────────────────────────────────┘  │    │
│   │                                                                 │    │
│   │   Context Relevance (30%)                                    │    │
│   │   ├─────────────────────────────────────────────────────────┤  │    │
│   │   │ Is this relevant to user's current context?             │  │    │
│   │   │ Hotel guest asking about gym: Fitness + Hospitality     │  │    │
│   │   └─────────────────────────────────────────────────────────┘  │    │
│   │                                                                 │    │
│   │   History Accuracy (25%)                                     │    │
│   │   ├─────────────────────────────────────────────────────────┤  │    │
│   │   │ How often has this agent succeeded with similar cases?  │  │    │
│   │   │ Culinary Expert success rate: 92%                        │  │    │
│   │   └─────────────────────────────────────────────────────────┘  │    │
│   │                                                                 │    │
│   │   Load Factor (10%)                                          │    │
│   │   ├─────────────────────────────────────────────────────────┤  │    │
│   │   │ Is the agent available and responsive?                   │  │    │
│   │   │ High load = lower score                                  │  │    │
│   │   └─────────────────────────────────────────────────────────┘  │    │
│   │                                                                 │    │
│   └─────────────────────────────────────────────────────────────────┘    │

│   ┌─────────────────────────────────────────────────────────────────┐    │
```
