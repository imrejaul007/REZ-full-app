# Karma System

## Overview

The Karma System is ReZ's social responsibility initiative that allows users to contribute to charitable causes and community impact projects. Users earn "Karma Points" through socially positive actions, creating a virtuous cycle of giving and receiving rewards.

---

## Features

### Karma Points

Unlike regular loyalty points, Karma Points are earned through ethical actions and charitable contributions. These points can be redeemed for rewards while also making a real-world impact.

#### Earning Karma Points

| Action | Karma Points | Description |
|--------|--------------|-------------|
| Order completion | 1 KP per Rs. 100 spent | Automatically earned on every order |
| Eco-friendly delivery choice | 10 KP | Opt for sustainable packaging |
| Leaving honest review | 5 KP | Verified purchase reviews only |
| Reporting issues/bugs | 20 KP | Help improve the platform |
| Referring new users | 50 KP | First successful referral |
| Participating in surveys | 15 KP | Product and service feedback |
| Festival charity donation | 2x multiplier | Double Karma on donation events |
| Zero-contact delivery | 5 KP | Reduce environmental impact |
| Recycling packaging | 25 KP | Upload photo proof of recycling |
| Supporting local businesses | 3 KP per Rs. 100 | Extra bonus for local merchants |

#### Karma Point Conversion
- 100 Karma Points = Rs. 5 credit (lower rate than loyalty points)
- Minimum redemption: 500 Karma Points
- Karma Points never expire
- Cannot be purchased or transferred

## User Guide

### Accessing Karma Features

1. Open the ReZ app
2. Tap on "Karma" tab in bottom navigation
3. View your Karma Points, impact dashboard, and challenges

### Earning Your First Karma Points

1. Make any purchase on ReZ
2. Points are automatically credited after order delivery
3. Go to Karma tab to view earned points
4. Allocate points to your chosen cause

### Allocating Points to Causes

1. Navigate to Karma > "My Causes"
2. View available charitable causes
3. Use sliders to allocate your Karma Points
4. Confirm allocation
5. Receive impact updates via notifications

### Completing Challenges

1. Go to Karma > "Challenges"
2. View active weekly and monthly challenges
3. Complete required actions
4. Claim reward when challenge is fulfilled
5. Points are credited within 24 hours

### Viewing Impact Reports

1. Tap on Karma > "My Impact"
2. View real-time impact statistics
3. See breakdown by cause
4. Download impact report (monthly PDF)

## Technical Implementation

### Karma Point Calculation

```javascript
// Karma points earned per order
const calculateKarmaPoints = (orderTotal, options = {}) => {
    const {
        isEcoFriendlyDelivery = false,
        isZeroContactDelivery = false,
        isLocalMerchant = false
    } = options;

    let points = Math.floor(orderTotal / 100); // Base: 1 KP per Rs. 100

    // Bonus for eco-friendly choices
    if (isEcoFriendlyDelivery) {
        points += 10;
    }

    // Bonus for zero-contact delivery
    if (isZeroContactDelivery) {
        points += 5;
    }

    // Extra bonus for local merchants
    if (isLocalMerchant) {
        points += Math.floor(orderTotal / 100) * 2; // 3 KP per Rs. 100
    }

    // Festival multiplier check
    const festivalMultiplier = checkFestivalMultiplier();
    points = Math.floor(points * festivalMultiplier);

    return points;
};
```

### Cause Allocation Processing

```javascript
const allocateToCauses = async (userId, allocations) => {
    const user = await db.users.findOne({ _id: userId });

    // Validate total allocation doesn't exceed available
    const totalAllocated = Object.values(allocations).reduce((a, b) => a + b, 0);

    if (totalAllocated > user.karmaPoints) {
        throw new Error('Insufficient Karma Points');
    }

    // Create allocation record
    const allocationRecord = {
        userId,
        allocations,
        totalAllocated,
        createdAt: new Date()
    };

    // Update user's cause balances
    for (const [causeId, amount] of Object.entries(allocations)) {
        await db.causeAllocations.updateOne(
            { causeId },
            {
                $inc: { totalAllocated: amount },
                $push: { userAllocations: { userId, amount, date: new Date() } }
            }
        );
    }

    // Deduct from user's Karma Points
    await db.users.updateOne(
        { _id: userId },
        {
            $inc: { karmaPoints: -totalAllocated },
            $push: { karmaHistory: allocationRecord }
        }
    );

    // Send notification to user
    await sendImpactNotification(userId, allocations);

    return allocationRecord;
};
```

Returns list of available charitable causes.

### POST /api/karma/allocate
Allocate Karma Points to specific causes.

**Request:**
```json
{
    "allocations": {
        "education": 500,
        "environment": 1000,
        "healthcare": 750,
```

Returns active challenges and user's progress.

### POST /api/karma/challenges/:id/claim
Claim reward for completed challenge.

Returns karma leaderboard (weekly/all-time).

**Query Parameters:**
- `period` (string): 'weekly' or 'alltime' (default: 'weekly')
- `limit` (number): Number of results (default: 50)

## Troubleshooting

### Karma Points Not Credited
1. Verify order was delivered (points credit after delivery)
2. Check if delivery options were eco-friendly/zero-contact
3. Points may take up to 24 hours to appear
4. Contact support with order ID

### Cause Allocation Failed
1. Ensure you have sufficient Karma Points
2. Minimum allocation per cause is 50 KP
3. Allocations are processed immediately
4. Contact support if deduction didn't reflect

### Challenge Not Progressing
1. Verify action was performed during active challenge period
2. Some challenges require specific delivery options
3. Check challenge requirements carefully
4. Clear app cache if progress doesn't update

## Frequently Asked Questions

**Q: Can I convert regular loyalty points to Karma Points?**
A: No, Karma Points are earned only through socially positive actions and cannot be converted from loyalty points.

**Q: Can I get a refund of Karma Points allocated to causes?**
A: No, Karma Point allocations are final and non-refundable as they represent committed donations.

**Q: How do I know my Karma Points made a real impact?**
A: We send monthly impact reports with verified data from our NGO partners showing exactly how contributions were used.

**Q: Do Karma Points expire?**
A: No, Karma Points never expire. They remain in your account until you allocate them to causes.

**Q: Can I see which specific projects my donations supported?**
A: Yes, you can view detailed project updates in the Karma tab under "My Impact".
